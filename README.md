
## 앱 개요
```
기능 : 구글 지도를 활용해서 지도 UI를 구현합니다.
       현재 위치로 이동 버튼을 눌러서 GPS 상의 현재 위치로 이동합니다.
       출근 가능한 위치로 이동하면 [출근하기] 버튼을 눌러서 출근 체크를 할 수 있습니다.
       출근 가능한 위치가 아니라면 [출근하기] 버튼은 보이지 않습니다.
       
사용한 플러그인 : google_maps_flutter, geolocator
```
